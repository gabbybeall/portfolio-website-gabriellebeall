HTML . CSS . JavaScript

This world clock application was built with HTML, CSS, and advanced JavaScript using Moment.js and provides users with real time displays of current dates and times across various global locations. Using the dropdown menu, you can select one of the cities listed or click 'My current timezone' to see the current date and time for your location.
